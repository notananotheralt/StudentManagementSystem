# StudentManagementSystem
School Assignment

You input the student's name and age. After that you hit add button in order to add the student. You then click view students to see the ids of existing students. You click view courses to see created courses. You click add courses and fill it in. In order to see information about a student, you put in their ID, and then click Student Information. In order to put that student in a class, you keep their ID in that student Id textbox and select a course, then hit enroll student to enroll them. You hit save data and load data to interact with the csv files. You have to create a course before you can select one, even if you load data.
